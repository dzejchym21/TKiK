# TKiK

### Tabela Tokenów
| Kod Tokena | Przykładowa Wartość | Opis |
| :--- | :--- | :--- |
| **INT** | `123`, `5` | Liczba całkowita (ciąg cyfr). |
| **ID** | `x`, `wynik1` | Identyfikator (litera, potem litery/cyfry). |
| **PLUS** | `+` | Operator dodawania. |
| **MINUS** | `-` | Operator odejmowania. |
| **MUL** | `*` | Operator mnożenia. |
| **DIV** | `/` | Operator dzielenia. |
| **LPAREN** | `(` | Nawias otwierający. |
| **RPAREN** | `)` | Nawias zamykający. |
| **ERROR** | `@`, `$` | Nieoczekiwany znak (błąd leksykalny). |
| **EOF** | `None` | Koniec wyrażenia (End of File). |
